## Image Requirements:
- Format: PNG
- Recommended size: 1200x800px or similar 3:2 ratio
- 2x resolution for retina displays
- Dark theme screenshots to match the courses page
- Actual app UI, not mockups

## How to Take Screenshots:
1. Use Chrome DevTools device mode
2. Set to MacBook Pro (1440x900)
3. Take screenshots at 2x resolution
4. Crop to just the main content area
5. Save as PNG in this directory

Once you add the 3 images, the courses page will automatically display them.
